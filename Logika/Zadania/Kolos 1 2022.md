
###### 1. Które z poniższych zdań są równoważne zdaniu $\sim p \vee q$?

a) $(q\downarrow (p\downarrow p))  \mid (q\downarrow (p \downarrow p))$

$$\begin{gathered}

\sim(q\vee (p\downarrow p))\mid \sim (q\vee (p\downarrow p))\\
\sim(q\vee \sim (p\vee p))\mid \sim (q\vee \sim (p\vee p))\\
\sim(q\vee \sim p)\mid \sim (q\vee \sim p)\\
\sim [\sim(q\vee \sim p)\land \sim (q\vee \sim p)]\\
(q\vee \sim p)\vee (q\vee \sim p)\\
q\vee \sim p\\
\sim p \vee q
\end{gathered}$$
:LiCheck:

b) $(p\mid p)\downarrow q$

$$\begin{gathered}
\sim (p\land p) \downarrow q\\
\sim p \downarrow q\\
\sim (\sim p \vee q)\\\\
p \land \sim q
\end{gathered}$$

:LiCross:

c) $p\mid (q\downarrow q)$

$$\begin{gathered}
p\mid \sim (q\vee q)\\
p \mid \sim q\\
\sim (p\land \sim q)\\
\sim p\vee q
\end{gathered}$$

:LiCheck:

###### 2) Które z podanych wyrażeń rachunku zdań jest tautologią?

a) $p\implies (q\implies ((p\land q)\implies p))$

To musimy sprawdzić jedynie czy kiedykolwiek będzie fałszywe, a implikacja jest fałszywa jedynie kiedy z prawdy wynika fałsz.

$$\begin{gathered}
1\implies 0\\
p=1\\
(q\implies ((p\land q)\implies p)) = 0\\
\downarrow\\
q=1\\
((p\land q)\implies p)=0\\
podstawiając \\
1\implies 1 = 1\neq 0
\end{gathered}$$

Dane prowadzące do fałszu prowadzą do sprzeczności.

Jest to tautologia.

b) $((p\vee q)\land \sim p)\implies \sim q$

$$\begin{gathered}
\sim q = 0\\
q=1\\
((p\vee q)\land \sim p)=1\\
(\sim p \land (p\vee q))=1\\
(\sim p \land p) \ \vee (\sim p \land q)=1\\
(\sim p \land q)=1\\
\sim p=1\\
p = 0\\
\text{Podstawiajac dla sprawdzenia:}\\\\
((0\vee 1)\land \sim 0)\implies \sim 1\\
(1\land 1)\implies 0\\
1\implies 0
\end{gathered}$$

To nie jest tautologia bo dla $q=1, p=0$ zdanie fałszywe

c) $(p\mid p)\oplus (q\land \sim q)$

$$\begin{gathered}
(p\mid p)\Leftrightarrow \sim (p\land p)\Leftrightarrow\sim p \\\\
q\land \sim q =0\\\\
a\oplus b = 0 \Leftrightarrow a=b=1 \vee a=b=0\\
\sim p =0\\
p = 1
\end{gathered}$$

Dla $p=1$ fałsz.

###### 5) Formuła $\alpha$ rachunku zdań jest postaci: $(p\land q)\vee \sim (\sim q\implies p)$. Odpowiadająca jej koniunkcyjna postac normalna to:

$$\begin{gathered}
a)\  (p\land q)\vee (\sim p \land \sim q)\\
b) \ (p\vee \sim q)\land (q\vee \sim p)\\
c) \ p\land \sim q
\end{gathered}$$

$$\begin{gathered}
(p\land q)\vee \sim (\sim q\implies p)\\
 (\sim q\implies p)\Leftrightarrow\sim (\sim q\land \sim p)\Leftrightarrow q\vee p\\\\
 (p\land q)\vee \sim (q\vee p)\\
 (p\land q)\vee (\sim q \land \sim p)\\
 ((p\land q)\vee \sim q )\land ((p\land q) \vee \sim q)\\
 (p\vee \sim q)\land (q\vee \sim p)
\end{gathered}$$
